Appz is a web development toolkit that handles the full lifecycle — from project scaffolding to deployment.

## What is Appz?

Appz is a single CLI that replaces a fragmented toolchain:

- **`appz init`** — scaffold projects from templates (WordPress, Astro, Next.js, etc.)
- **`appz dev`** — framework-aware dev server with DDEV for PHP/CMS projects
- **`appz build`** — optimized production builds
- **`appz deploy`** — deploy to any platform
